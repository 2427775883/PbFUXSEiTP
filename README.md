# 项目简介

## 核心功能
- 数据清洗：包括缺失值处理和异常值处理
- 数据预处理：包括标准化处理
- 数据统计
- 数据可视化
- 研究结论：数据集来源于和鲸社区

## 适用场景
- 天文观测数据分析
- 行星分布、性质和形成机制研究
- 行星系统形成和演化研究
- 外星生命可能性研究
- 未来太空探索参考信息

## 技术特点
- 使用 Python 语言进行编程
- 基于 pandas 和 numpy 库进行数据处理
- 使用 matplotlib 进行数据可视化
